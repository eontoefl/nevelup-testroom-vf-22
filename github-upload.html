📋 교체 방법

  1. 아래 2개 파일을 "다운로드" 클릭
  2. GitHub 레포 → 해당 파일의 경로로 이동
  3. 파일 클릭 → ✏️ 연필 아이콘 (Edit this file) 클릭
  4. Ctrl+A → Delete (전체 삭제)
  5. 다운로드 받은 파일을 VS Code나 메모장으로 열기
  6. Ctrl+A → Ctrl+C (전체 복사)
  7. GitHub 편집기에서 Ctrl+V (붙여넣기)
  8. Commit changes 클릭
